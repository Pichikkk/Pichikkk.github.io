Задание 2. Взаимодействие с пользователем

Задание 2.1: Переменные admin и name

Объявите две переменные: admin и name. Запишите в name строку "Валерия Пичикова". Скопируйте значение из name в admin. Выведите admin.

Код решения:

function task2_1() {
    const name = "Валерия Пичикова";
    const admin = name;
    document.getElementById('task2_1Result').textContent = admin;
}

Задание 2.2: Подтверждение возраста

Скрипт спрашивает возраст и после ввода уточняет его. Если нажать "Отмена", скрипт просит ввести возраст снова.

Код решения:

function task2_2() {
    let age;
    do {
        age = prompt("Сколько вам лет?", "");
    } while (age === null || age === "");
    
    let isConfirmed = false;
    while (!isConfirmed) {
        isConfirmed = confirm(`Вы уверены, что вам ${age} лет?`);
        if (!isConfirmed) {
            age = prompt("Сколько вам лет?", "");
        }
    }
    document.getElementById('task2_2Result').textContent = 
        `Подтвержденный возраст: ${age} лет`;
}
